# Galerie de scientifiques étonnants

La galerie de scientifiques étonnants est une application React qui affiche une liste d'images de scientifiques remarquables.

## Description

Cette application présente une liste d'images de scientifiques remarquables, notamment Katherine Johnson, une mathématicienne américaine dont les calculs étaient indispensables pour le programme Mercury de la NASA.

## Fonctionnalités

- Affichage d'une liste d'images de scientifiques.
- Utilisation de composants React pour organiser et afficher les données.

## Installation

1. Clonez ce dépôt :

```bash
git clone https://github.com/Paul-FranckDencausse/gallery.git
```

2. Ouvrez le projet dans votre éditeur de code.

3. Exécutez `npm install` pour installer les dépendances.

4. Exécutez l'application avec `npm start`.
